# Cresent Clinic Management System

A web-based hospital and clinic management system designed to streamline medical practice operations, patient tracking, staff management, and clinical reporting.

## Features

- **Patient & Receptionist Portal**: Easy check-in, registration, and scheduling.
- **Doctor & Staff Management**: Track staff schedules, roles, and assignments.
- **Inventory & Suppliers**: Manage clinic supplies, inventory tracking, and vendor contact info.
- **Reporting & PDF Generation**: Generates automated PDFs for reports using FPDF.
- **WhatsApp Notification Integration**: Message dispatch system built into the app.
- **Database Backups**: Automates cron-based database backups.
- **Storage System**: Integration with external storage (Supabase).

## Tech Stack

- **Backend**: PHP (Custom router)
- **Database**: MySQL / MariaDB
- **Frontend**: HTML, CSS, JavaScript (Vanilla design)
- **Utilities**: Composer dependencies, Supabase Storage integration, FPDF
